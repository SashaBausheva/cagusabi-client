About The Project & The Engineers

The Project

This application originated as a team assignment in which a group of General Assembly's Software Engineering Immersive engineers were to build an app that allows users to upload images onto a virtual file system. All users can view/download images, whereas owners can also remove and edit the images they own.

The project you are seeing now is a forked and improved version by Sasha Bausheva.

You can find the source code on Sasha's GitHub page.
